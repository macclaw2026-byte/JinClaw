# Marketing Improvement Loop

Use the shared monitoring standard from `../../self-cognition-orchestrator/references/monitoring-and-retrofit-framework.md` when reviewing performance.

After each real marketing task, review:

1. Which segment was targeted?
2. Which channels were chosen?
3. What message angle was used?
4. What conversion event was targeted?
5. What evidence suggests it worked or failed?
6. Which monitor produced the strongest signal?
7. Which monitor was missing, weak, or noisy?
8. What should change next time?

## Outputs

- short learning note
- proposed adjustment
- whether the core skill should be updated
- whether a niche extension should be created
- whether a shared monitoring retrofit is needed

## Good reasons to create an extension skill

- the same niche appears repeatedly
- buyer behavior is noticeably different from the generic model
- message angles become vertical-specific
- website conversion logic becomes category-specific
- the same monitoring gaps recur in that niche
