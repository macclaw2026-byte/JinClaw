# Standard Deliverables

Use the shared monitoring standard from `../../self-cognition-orchestrator/references/monitoring-and-retrofit-framework.md` when presenting recommendations.

When using `us-market-growth-engine`, prefer concrete deliverables over vague advice.

## Common deliverables

- ICP summary
- buyer segment map
- customer pain-point map
- channel strategy
- outreach sequence
- CTA recommendations
- landing-page recommendations
- niche keyword set
- website conversion checklist
- measurement plan
- next-iteration plan

## Minimum useful output

For a real task, try to produce at least:

1. target segment definition
2. conversion path
3. channel recommendation
4. messaging angle
5. next action plan
6. confidence / blind spots / compensating control
