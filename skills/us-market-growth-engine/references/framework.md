# US Market Growth Framework

Use the shared monitoring standard from `../../self-cognition-orchestrator/references/monitoring-and-retrofit-framework.md` so growth planning names its evidence quality, backstops, and blind spots.

## Universal sequence

1. Define what is being sold
2. Define who buys it
3. Define why they buy it
4. Define where they can be reached
5. Define what action you want them to take
6. Define what proof builds trust
7. Launch a small focused motion
8. Measure response and improve

## Buyer types

### B2B
- trade buyers
- designers
- contractors
- builders
- procurement roles
- distributors / wholesalers

### B2C
- end consumers
- homeowners
- style-driven buyers
- price-sensitive buyers
- premium buyers

## Core outputs this skill should be able to produce

- ICP summary
- segment map
- channel strategy
- outreach plan
- website conversion recommendations
- message angles
- CTA recommendations
- feedback loop improvements

## Monitoring lens

For each output, identify:
- strongest current signal
- likely weak link
- compensating control or delayed review if confidence is limited
