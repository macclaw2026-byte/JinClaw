# Conversion Path Design

## Path types

### Direct response path

- outreach -> response -> meeting / quote -> close

### Quote path

- outreach -> website -> quote request -> sales handling -> close

### Partner path

- outreach -> partner application -> qualification -> onboarding

### Education-to-conversion path

- outreach / content -> site education -> inquiry / signup -> close

## Per-path fields

- path_id
- path_type
- first_CTA
- second_CTA
- success_event
- fallback_action
- stop_condition
- owner_type
