# Marketing Strategy Engine Segmentation And Strategy

## Segmentation dimensions

### Account tier

- A / B / C / D from lead scoring

### Persona type

- founder
- buyer
- operator
- designer
- contractor
- partner
- distributor

### Buying context

- active project
- exploratory
- partnership fit
- reactivation fit
- weak-signal observation

### Channel readiness

- email-ready
- form-first
- social-first
- manual-only

## Strategy object

Each strategy object should include:

- strategy_id
- target_scope
- primary_angle
- support_angle
- anti_angle
- value_map
- proof_map
- CTA
- fallback_CTA
- channel_recommendation
- risk_level
- approval_required
