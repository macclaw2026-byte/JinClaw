# Field completion rules

## Category
Choose the most specific available category that matches the fixture type.
Common mappings:
- chandelier / lantern chandelier / wagon-wheel chandelier -> Ceiling Lights
- pendant -> Pendant Lights
- wall sconce -> Wall Lights

## Shipping template
If a known valid saved template exists, prefer that over creating a new one.
Example already seen in this workflow:
- `Free shipping (Free Shipping)`

## Packing unit
A listing is not ready if it still says `No packing units added yet`.
At minimum create `Packing Unit #1` with:
- Unit Type: Box
- Quantity: 1
- Length/Width/Height: defensible values based on listing size clues, package norms, or existing sibling listings

## When uncertain
If exact values are unavailable:
1. inspect title, dimensions, and existing images
2. inspect sibling listings of similar products in the seller center when accessible
3. use a conservative, plausible packaging estimate
4. if uncertainty remains material, ask the user briefly before submission
