# Batch checklist

For each uploaded batch:

1. open Bulk Import
2. if using Excel, choose the correct mapping template and upload the file
3. if using API-connected import, trigger the prepared import submission from Bulk Import
4. wait for imported products to appear in Listings as drafts

For each draft listing:

1. confirm status is DRAFT
2. inspect product description and clean garbled text if present
3. inspect current image count
4. confirm whether first 3 are scene images
5. generate/upload missing scene images until >=3
6. verify first 3 are scene images
7. set category
8. set shipping template
9. add packing unit
10. save and re-check page state
11. submit for review
12. verify status moved out of DRAFT
13. record result

## Result labels
- submitted
- blocked-image-order
- blocked-upload
- needs-user-input
- blocked-required-field
