# Feedback Loop

## Feedback destinations

### Back to prospect data

- invalid endpoint
- dead company
- wrong role
- wrong entity
- duplicate entity

### Back to strategy

- weak angle
- wrong CTA
- wrong channel
- objection cluster
- stronger persona discovered

## Reporting outputs

### Daily

- attempts
- successful sends/submissions
- replies
- unsubscribes
- invalids
- top positive segments

### Weekly

- segment conversion
- channel win rate
- strategy win rate
- data quality drift

### Exceptions

- bounce spike
- suppression spike
- complaint-risk event
- channel degradation
