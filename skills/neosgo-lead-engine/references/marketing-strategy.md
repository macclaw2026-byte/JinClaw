# Neosgo 潜客营销策略

## 目标
把本地高分潜客池转化成可持续成交机会，优先推动：
- 网站注册 / trade account 申请
- 报价询盘
- 索样
- 预约沟通
- 线下项目合作或首单成交

## 核心原则
1. 分群而不是群发
2. 先价值后成交
3. 先小转化后大转化
4. 每次触达都可追踪
5. 用真实反馈反推模型与文案

## 细分人群与价值主张

### 1. 室内设计 / 建筑设计
**目标角色**
- Owner / Principal / Founder
- Principal Designer / Interior Designer / Architect

**主打价值**
- trade pricing
- 快速报价
- 项目型选品支持
- 样品/方案协作
- 更高审美一致性与客户成交率

**推荐 CTA**
- 预约 15 分钟项目匹配沟通
- 提交当前项目需求拿报价
- 获取 trade catalog / sample policy

### 2. Builder / Contractor / Remodeler
**目标角色**
- Owner
- Project Manager
- Purchasing / Procurement
- Estimator

**主打价值**
- 稳定供货
- 项目配套能力
- 批量价格
- 交付与补货效率
- 售后与协调响应

**推荐 CTA**
- 提交项目清单
- 获取批量报价
- 预约供应适配沟通

### 3. Electrician / Lighting installer
**目标角色**
- Owner
- Lead Electrician
- Operations Manager

**主打价值**
- 安装友好
- 快速补货
- 适合转介绍或工程打包
- 降低采购摩擦

**推荐 CTA**
- 查看适合安装端的产品清单
- 讨论合作/推荐机制

### 4. Furniture retailer / showroom / kitchen & bath
**目标角色**
- Owner
- Buyer
- Merchandising / Purchasing

**主打价值**
- 差异化产品线
- 批发利润空间
- 陈列和样品合作
- 稳定供货与复购支持

**推荐 CTA**
- 申请批发目录
- 申请样品
- 预约渠道合作沟通

### 5. Property / Hospitality / Realtor
**目标角色**
- Property Manager
- Broker / Team Lead
- Hospitality procurement

**主打价值**
- 样板房/翻新/项目采购
- 推荐与项目线索引导
- 批量更新与场景化配套

**推荐 CTA**
- 提交当前项目需求
- 预约方案/报价沟通

## 渠道优先级
1. 邮件
2. 网站承接页 / 表单
3. 电话 / WhatsApp（后续）
4. LinkedIn（后续）

## 开发节奏（默认邮件）
- Day 1：首封价值型触达
- Day 3：补充案例/产品利益点
- Day 7：跟进提醒
- Day 12：最后一次低打扰跟进

## 最小追踪指标
- sent
- delivered / bounced（若可得）
- opened（若可得）
- clicked（若可得）
- replied
- qualified
- quote_requested
- sample_requested
- meeting_booked
- won / lost

## 优化方法
- 按 segment_primary 看回复率
- 按 title 看询盘率
- 按州 / 城市看转化率
- 按文案 variant 看回复率
- 按触达节奏看后续行动率

## 风险控制
- 先小批量发送
- 有退订或最少合规说明
- 控制域名信誉
- 不做高频无差别群发
