# Daily Report Schema

For each daily product-selection report, try to include:

```text
report_date:
timezone:
report_goal:
new_products:
refined_products:
unmet_demand_to_product_paths:
marketplace_fit:
confidence:
why_new_today:
non_repeat_check:
notes:
```

## Per-product fields
- product_name
- source_mix
- why_it_matters
- current fit (Amazon / Walmart / other)
- existing product match
- improvement path
- new product path if relevant
- confidence
- competitor link when practical
```
