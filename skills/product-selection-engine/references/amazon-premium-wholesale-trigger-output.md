# Trigger and Output Logic

## Trigger
- default daily scheduled run at 20:00 America/New_York
- manual rerun allowed

## Anti-repeat logic
Suppress a candidate if it appeared recently and no meaningful delta exists.

Allow resurfacing only when:
- new evidence materially improves confidence
- a more precise sub-niche is now identified
- a stronger competitor gap or tool-gap makes the idea meaningfully different
- a prior weak candidate now has stronger proof

## Daily output categories
Each daily output should classify candidates as:
- new opportunity
- refined opportunity
- unmet-demand -> product path

## Minimum output for Amazon sheet
- at least 20 candidates
- each must include competitor link
- each must include reason, range, grade, survivability, recommendation
