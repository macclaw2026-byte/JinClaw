# Routing Map

## Route to product-selection-engine when data is about:
- demand
- trend strength
- marketplace activity
- competitor density
- review/rating depth
- price bands
- product structure
- manufacturability
- innovation inspiration

## Route to us-market-growth-engine when data is about:
- buyer segments
- value propositions
- competitor messaging
- landing-page structure
- conversion friction
- pain points
- content/channel clues

## Route to resilient-external-research when data is about:
- missing fields
- blocked sources
- conflicting evidence
- weak/partial extraction

## Route to safe-learning-log when data reveals:
- recurring extraction noise
- recurring branch mistakes
- repeated misleading fields
- a new high-value source pattern

## Route to runtime-evolution-loop when data reveals:
- repeated need for a new branch
- repeated routing mismatch
- repeated schema gaps
- recurring cross-skill improvement opportunities
