# Crowdfunding / Blue-Ocean Branch

Use this branch for Kickstarter, Indiegogo, and similar innovation/product-launch surfaces.

## Goal

Extract early product signals without confusing crowdfunding success with direct marketplace fit.

## Good source surfaces
- crowdfunding project pages
- category discovery pages
- product-launch / innovation showcase pages

## Recommended tool stack
- primary: web_fetch or local Crawl4AI
- backup: browser
- backup: resilient-external-research

## Key fields
- product concept
- problem statement
- feature innovation
- pricing signal
- social proof / backing signal when public
- delivery/manufacturing complexity clues
- what selling point seems most compelling

## Downstream targets
- product-selection-engine
- runtime-evolution-loop
- safe-learning-log

## Important rule
Use this branch mainly for inspiration, problem/solution framing, and feature abstraction — not as a direct replacement for marketplace demand validation.
