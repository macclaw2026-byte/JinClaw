# Competitor Page Branch

Use this branch for direct competitor website or landing-page analysis.

## Goal

Extract structured signals about positioning, offer design, conversion strategy, and message angle.

## Good source surfaces
- homepage
- product page
- collection/category page
- landing page
- pricing page
- about / proof / FAQ sections

## Recommended tool stack
- primary: web_fetch or local Crawl4AI
- backup: browser
- backup: guarded-agent-browser-ops when structure/interaction matters

## Key fields
- headline / primary message angle
- target customer clues
- offer structure
- pricing presentation
- trust/proof elements
- CTA patterns
- conversion friction / missing proof

## Downstream targets
- us-market-growth-engine
- product-selection-engine when the page also reveals useful product/market clues
