# Walmart Marketplace Branch

Use this branch for Walmart-oriented product-selection or broad-market utility analysis.

## Goal

Collect structured public evidence that helps infer:
- mainstream demand
- practical mass-market fit
- visible activity signals such as in carts / bought since yesterday when shown
- price/value ladder
- competition survivability

## Good source surfaces
- search results pages
- category pages
- product pages when publicly accessible

## Recommended tool stack
- primary: browser or local Crawl4AI
- backup: guarded-agent-browser-ops
- backup: resilient-external-research + public alternatives

## Key fields
- query/product phrase
- titles surfaced near the top
- prices
- in-cart / bought-since-yesterday style signals when visible
- repetition of value/private-label style offers
- category utility fit

## Downstream targets
- product-selection-engine
- resilient-external-research
- safe-learning-log when repeated anti-bot or visibility issues distort results
